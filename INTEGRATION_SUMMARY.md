## Как использовать

1. Откройте `index.html` в браузере
2. Заполните Модуль 1 (Настройки распоряжения)
   - Выберите тип проверки (Подготовка/Проведение)
   - Выберите тип распоряжения (Новое/Изменение)
   - Выберите уровень (ЦА/ТБ)
3. Заполните Модуль 2 (Опросник) - форма зависит от настроек Модуля 1
4. Заполните Модуль 3 (Команда проверки)
5. При необходимости заполните Модули 4 и 5
6. Нажмите кнопку **"Сформировать документ"**

## Проверка перед генерацией

Генератор автоматически проверяет:
- ✅ Все ли модули сохранили данные
- ✅ Выбран ли тип проверки в Модуле 1
- ✅ Заполнен ли Опросник в Модуле 2

При ошибке выводится alert с указанием проблемы.

## Структура данных

Данные передаются через ConfigManager:
```javascript
{
  SettingsOrder: { auditType, orderType, levelOrder, selectedTB },
  Questionnaire: { auditName, dates, basisPoint, ... },
  TeamAudit: { curators, leader, deputy, control, signer, teamMembers },
  ProcessesModule: { items, include, exclude },
  AutomatedSystems: { items }
}
```

## Генерация DOCX

После нажатия кнопки:
1. Собираются данные из всех модулей
2. Преобразуются в формат для docxtemplater
3. Загружается соответствующий шаблон
4. Генерируется документ и скачивается как `Распоряжение_Task_YYYY-MM-DD_HH-MM.docx`
